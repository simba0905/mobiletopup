This plugin for WooCommerce works with ThailandTopup.com API and allows anyone to send topup to a Thai pre-paid SIM with a WooCommerce virtual product.


== Description ==

[ThailandTopup.com](https://thailandtopup.com/) is a mobile phone credit portal for Thailand and allows you to easily top up a Thai prepaid mobile phone.
Use this plugin with WooCommerce to create a virtual product that anyone can buy to top up their own Thai phone.


== Installation ==

1. Upload the `mobiletopup` folder to the `/wp-content/plugins/` directory or install directly through the plugin installer
2. Activate the plugin through the 'Plugins' menu in WordPress or by using the link provided by the plugin installer
3. Visit the settings page in the Admin at `Settings -> ThailandTopup` and configure the plugin with your account details
4. Go to WooCommerce and create an new virtual product.
5. Choose the THB credit amount in the right side box.
6. Make a test with phone number "0800000004"
